# Lazy Corner

Lazy Corner is a **Single-Page Application (SPA)** built with **React + Vite**.  
It brings together a variety of fun, lightweight **public APIs** (like JokeAPI and Dog CEO API) to deliver different types of content from one place — jokes, images, and more.

## 🚀 Getting Started

To run the project locally:

1. Clone this repo
2. Run `npm install`
3. Run `npm run dev`
